# csci3081-cmake
Makefile options
```
make build //  Build and compile the project

make rebuild //  Clean up and rebuild the project

make run //  Run the project after build

make debug //  Debug the project after build

make clean //  Remove the build dir

make docs //  Generate doxygen

make lint //  Runs cpplint on project

make lintQ //  A faster version of 'make lint'

make venv //  Initialize requirements for the project (cpplint, clang-format, etc)
```
